The Temporal module

The API defined in this specification is exposed as a synthetic module with the specifier `"std:temporal"`. This is achieved through the modification to HostResolveImportedModule abstract operation in this section.

HostResolveImportedModule ( _referencingModule_, _specifier_ )

The implementation-defined HostResolveImportedModule abstract operation must additionally conform to the following requirement:

If _specifier_ is `"std:temporal"`, the operation must return the Module Record created by the following steps.

1. Let _objects_ be « %CivilDate%, %CivilTime%, %CivilDateTime%, %Instant%, %ZonedInstant% ». 1. Let _exports_ be « "CivilDate", "CivilTime", "CivilDateTime", "Instant", "ZonedInstant" ». 1. Let _realm_ be the current Realm Record. 1. Let _moduleRecord_ be ! CreateSyntheticModule(_exports_, the following steps, _realm_, *undefined*), with the following steps: 1. For each _object_ in _objects_, do 1. Let _id_ be ! Get(_object_, `"name"`). 1. Perform ! SetSyntheticModuleExport(_moduleRecord_, _id_, _object_). 1. Return _moduleRecord_.
